### Q3 (로그인 진입 트리거) — 답변 원문: `A + 마이페이지에 로그인 안되어있으면 로그인하세요 배너 상단`
→ **A 동작** (게이트된 액션 시도 시 자동 로그인 표시) + **마이페이지 화면 상단에 "로그인하세요" 배너 표시 (비로그인 시)** 라는 별도 FR로 분리해 캡처합니다.

### Q6 (V02 지역 범위) — 답변 원문: `우선 군산만 MVP로, 추후 변동가능성 있음`
→ **옵션 A (군산만)** 로 처리. "추후 변동" 부분은 비기능 제약 — 지역 데이터 모델/엔드포인트를 다중 지역으로 확장 가능하게 유지(하드코딩 회피).

### Q8 (세션 만료 처리) — 답변 원문: `로그인 토큰이 그렇게 짧지는 않을건데... 일단은 A로`
→ **옵션 A**로 처리. 토큰 수명 가정에 대한 별도 NFR은 추가하지 않음.
